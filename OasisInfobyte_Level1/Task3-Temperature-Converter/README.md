# Task 3: Temperature Converter

This is my **third task** from the Oasis Infobyte Level 1 Internship.  
It is a **web-based temperature converter** that converts between Celsius, Fahrenheit, and Kelvin.

---

## Features

- Converts temperatures between **Celsius, Fahrenheit, and Kelvin**.
- Interactive **input field** for entering temperature.
- **Dropdown selector** to choose the temperature unit.
- Displays conversion results instantly in a **result area**.
- Clean and minimalistic design for easy use.
- Built using **HTML, CSS, and JavaScript**.

---

## How to Run

1. Open the folder `Task-3-Temperature-Converter`.
2. Open `index.html` in your browser.
3. Enter a number in the **Degrees** field.
4. Select the unit (Celsius, Fahrenheit, Kelvin) from the dropdown.
5. Click **Convert** to see the result.

---

## Technologies Used

- **HTML5** – Structure of the web page
- **CSS3** – Styling and layout
- **JavaScript** – Logic for temperature conversion and interactivity
